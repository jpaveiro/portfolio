<h2>Pré-requisitos</h2>
Para rodar o projeto, é necessário apenas instalar a ferramenta Node.js v20+.

<h2>Rodar o projeto</h2>
Siga os passos a seguir:
<br><br>

<strong>- Passo 0: Navegue para a pasta que deseja salvar o projeto.<br></strong>
Clique com o botão direito dentro da pasta, e selecione "Abrir no terminal", como na imagem a seguir:
<img src="/images/open_terminal.png" style="width: 500px;">

<strong>- Passo 1: Clonar repositório</strong>
<br>
<code>git clone https://github.com/jpaveiro/portfolio</code>
<br><br>
<strong>- Passo 2: Navegue para o repositório</strong>
<br>
<code>cd portfolio</code>
<br><br>
<strong>- Passo 3: Instale as dependências utilizando npm</strong>
<br>
<code>npm install</code>
<br><br>
<strong>- Passo 4: Rode o projeto</strong>
<br>
<code>npm run dev</code>
<br><br>
Deve aparecer algo como:
<br><br>
<img src="/images/run_dev.png" style="width: 500px;">
<br>

#### O projeto está rodando. Agora é só acessar <code>http://localhost:3000</code> diretamente no navegador.
